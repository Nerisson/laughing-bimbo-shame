###################################################################################################

Description du contenu du dossier:

-Le dossier bin contient le programme compilé. 
-Le dossier src contient les fichiers sources. Dedans il y a:
	- Le dossier de l'API 
	- Les différents fichiers de définition de classes
	- Le fichier principal : main.cpp 
	- Le projet Qt Creator
-Le rapport en pdf

###################################################################################################

Attention: pour pouvoir lancer le programme, il est nécessaire d'avoir installé les libairies 
OpenGL et SDL. L'installation d'openGL est détaillée sur le wiki. Pour installer SDL, il suffit
de lancer la commande suivante dans un terminal:

sudo apt-get install libsdl*

###################################################################################################

Pour lancer le programme, aller dans le répertoire /bin et lancer ./openGL 
Déplacement de la caméra:
	- Changement d'angle de vue: clic gauche puis déplacer la souris
	- Déplacement dans le monde: 
		z: avancer
		s: reculer			
		q: aller à gauche
		d: aller à droite
		+: aller vers le haut 
		-: aller vers le bas
	- Sortir du programme : ESC
